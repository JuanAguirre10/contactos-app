# 📋 Contacts App

Aplicación monolítica de gestión de contactos construida con Node.js y Express.

## Tecnologías
- Node.js
- Express.js
- HTML / CSS / JavaScript (Vanilla)

## Funcionalidades
- ➕ Crear contacto
- 📋 Listar contactos
- ✏️ Editar contacto
- 🗑️ Eliminar contacto

## Instalación

```bash
npm install
```

## Ejecución

```bash
node server.js
```

Abrir en el navegador: http://localhost:3000
